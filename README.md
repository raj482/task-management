## database setup 
   * setup mysql database by using .env.example
## configration
   use .env.example for configration
   
## install project 
1. run commonds
   * npm install
   * composer install
2. migrate database with seeder
    * php artisan migrate
    * php artisan db:seed
3. run commond to run project 
    * php artisan serve
    * npm run dev
4. visit the localhost in browser click in login page
